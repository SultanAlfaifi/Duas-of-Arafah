# Arafah Duas | أدعية يوم عرفة

A clean, elegant, and fully static web app for reading selected duas and adhkar on the Day of Arafah through a smooth card-based reading experience.

تطبيق ويب بسيط وأنيق لقراءة مجموعة مختارة من أدعية يوم عرفة، يشمل أدعية قرآنية، وأدعية نبوية، وأذكارًا جامعة، من خلال تجربة قراءة سلسة تعتمد على البطاقات التفاعلية.

---

## English Description

**Arafah Duas** is a static web app designed to help users read and reflect on duas during the blessed Day of Arafah.  
The app presents supplications in clean, full-width cards with smooth scroll-based animations, progress tracking, a reading timer, dark mode, and helpful educational modals about the virtues of Arafah and the etiquettes of dua.

The project is built without any backend, database, login system, or external server. It can be hosted directly on **GitHub Pages**.

---

## الوصف العربي

**أدعية يوم عرفة** هو موقع بسيط وروحاني يساعد المستخدم على قراءة الأدعية والأذكار بطريقة مرتبة ومريحة في يوم عرفة.  
يعرض الموقع الأدعية داخل بطاقات أنيقة بتجربة تفاعلية سلسة، مع حفظ التقدم، واحتساب مدة الدعاء، ودعم الوضع الليلي، بالإضافة إلى نوافذ تعريفية عن فضل يوم عرفة وآداب الدعاء.

المشروع يعمل بالكامل بدون باك إند، وبدون تسجيل دخول، وبدون قاعدة بيانات، ويمكن رفعه مباشرة على **GitHub Pages**.

## Features | المميزات

### English

- Clean and elegant Islamic-inspired interface
- Fully responsive mobile-first design
- RTL support for Arabic content
- Smooth card-based dua reading experience
- Scroll-based progress tracking
- Reading timer that starts automatically
- Progress saved locally using LocalStorage
- Dark mode support
- Reset progress option
- Modal section for the virtues of the Day of Arafah
- Modal section for the etiquettes and sunnah of dua
- No backend required
- No database required
- Ready for GitHub Pages deployment

### العربية

- واجهة نظيفة وأنيقة بطابع روحاني مناسب ليوم عرفة
- تصميم متجاوب مناسب للجوال أولًا
- دعم كامل للغة العربية واتجاه RTL
- عرض الأدعية داخل بطاقات تفاعلية جميلة
- تتبع التقدم أثناء القراءة
- مؤقت تلقائي لحساب مدة الدعاء
- حفظ التقدم محليًا عبر LocalStorage
- دعم الوضع الليلي
- خيار إعادة البدء من جديد
- نافذة لعرض فضل يوم عرفة
- نافذة لعرض سنن وآداب الدعاء
- لا يحتاج إلى باك إند
- لا يحتاج إلى قاعدة بيانات
- جاهز للنشر على GitHub Pages
